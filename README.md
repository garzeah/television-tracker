## Movie Tracker

A web application that allows users to search for movies.

### Built With

- React
- Redux
- Node
- Express
- MongoDB

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/garzeah/movie-tracker.git
   ```
2. Install NPM packages in both root and client directory
   ```sh
   npm install
   ```
3. Create a .env file
4. Start the server
   ```sh
   npm start
   ```
